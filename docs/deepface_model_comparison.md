# DeepFace 模型 & 检测器选型指南

> 基于 DeepFace v0.0.100，结合 WCM 内容审核场景的选型分析。

## 识别模型（Model）

| 模型 | Embedding 维度 | LFW 准确率 | 推理速度 | 推荐度 |
|------|--------------|-----------|---------|-------|
| **Facenet512** | 512 | 99.65% | 中等 | ⭐⭐⭐⭐⭐ |
| Facenet | 128 | 99.20% | 快 | ⭐⭐⭐⭐ |
| ArcFace | 512 | 99.41% | 中等 | ⭐⭐⭐⭐ |
| SFace | 128 | 99.60% | **最快** | ⭐⭐⭐⭐ |
| GhostFaceNet | 512 | 99.53% | 快 | ⭐⭐⭐⭐ |
| VGG-Face | 4096 | 97.78% | 慢 | ⭐⭐ |
| DeepFace | 4096 | 97.35% | 慢 | ⭐⭐ |
| OpenFace | 128 | 93.80% | 快 | ⭐ |
| DeepID | 160 | 97.05% | 快 | ⭐⭐ |
| Dlib | 128 | 99.38% | 中等 | ⭐⭐⭐ |

### 关键结论

- **Facenet512** 在 LFW 上准确率最高（99.65%），512 维向量在精度和存储间取得良好平衡
- **SFace** 是速度最快的高精度模型（99.60%），仅 128 维，适合对速度敏感的场景
- **VGG-Face / DeepFace** 维度高达 4096，存储和搜索开销大，精度反而不如新模型，不推荐

## 人脸检测器（Detector Backend）

| 检测器 | 精度 | 速度 | 适用场景 |
|--------|------|------|---------|
| **retinaface** | ⭐⭐⭐⭐⭐ | 慢 | 精度优先，侧脸/小脸/遮挡 |
| **yolov8** | ⭐⭐⭐⭐ | **快** | 速度与精度平衡 |
| yunet | ⭐⭐⭐ | **最快** | 极致速度，资源受限 |
| mtcnn | ⭐⭐⭐⭐ | 中等 | 经典方案，稳定 |
| ssd | ⭐⭐⭐ | 快 | 正脸为主的场景 |
| opencv | ⭐⭐ | 快 | Haar 级联，最轻量但精度差 |
| mediapipe | ⭐⭐⭐ | 快 | 移动端优化 |
| centerface | ⭐⭐⭐ | 快 | 轻量级 |

### 关键结论

- **retinaface** 检测精度最高，尤其在侧脸、小脸、遮挡场景下表现突出，但速度最慢
- **yolov8** 是最佳性价比选择，速度比 retinaface 快 3-5 倍，精度损失很小
- **yunet** 最轻量最快，适合对延迟极敏感或资源受限的部署

## 推荐组合

| 场景 | Model | Detector | 理由 |
|------|-------|----------|------|
| **精度优先**（内容审核） | `Facenet512` | `retinaface` | 最高精度组合 |
| **速度与精度平衡** | `Facenet512` | `yolov8` | 检测速度快 3-5x，识别精度不变 |
| **极致速度** | `SFace` | `yunet` | 最快组合，精度仍有 99.6% |
| **低维向量（省存储）** | `Facenet` / `SFace` | `yolov8` | 128 维，搜索更快，存储减半 |

## WCM 项目当前配置

```env
WCM_DEEPFACE_MODEL=Facenet512
WCM_DEEPFACE_DISTANCE_METRIC=cosine
detector_backend=retinaface
```

**当前已是精度最优解。** 如需提速，建议仅将检测器从 `retinaface` 换为 `yolov8`，识别模型保持 `Facenet512` 不变。

## 距离阈值参考（Facenet512 + Cosine）

| Cosine Distance | 置信度 | 场景 |
|----------------|--------|------|
| ≤ 0.20 | 极高 | 几乎相同的照片 |
| ≤ 0.30 | 高（DeepFace 默认） | 标准人脸验证 |
| ≤ 0.35 | 中等 | 搜索/检索，容忍角度光照变化 |
| ≤ 0.40 | 宽松 | 候选召回，配合人工复核 |

> **注意**：当前 DeepFace v0.0.100 的 ANN 搜索固定返回 Euclidean 距离（`distance_metric` 参数无效）。
> Facenet512 Euclidean 默认阈值为 **23.56**。

## 搜索速度瓶颈分析

1. **人脸检测**（detector）— 换 `yolov8` 提速最明显
2. **特征提取**（model）— Facenet512 速度适中，无需更换
3. **向量搜索**（ANN 索引）— 万级数据量不是瓶颈，无需优化
